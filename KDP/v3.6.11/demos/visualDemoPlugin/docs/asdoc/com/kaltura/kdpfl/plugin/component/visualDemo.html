public function visualDemo(viewColor:Number, viewAlpah:Number)

Constructor. This function draws a rectangle with an X to show how plugins behave in terms of streaching and full screen. the 10X10 square will be stretched as the component streches. The parameters are passed from the config.xml file, as can be seen in visualDemoCode.initializePlugin. Remove them and replace with your own parameters if needed.

Parameters
viewColor:Number — color of created rectangle
 
viewAlpah:Number — alpha of created rectangle
