registerKDPCallbacks()method 
public function registerKDPCallbacks():void

This function does 1 of 2 possible things: 1. If the flashvar jsInterfaceReadyFunc is set to true, this function waits until the tag of the html page surrounding the player is ready before registering the KDP callbacks. It does so by starting the callbackTimer, and re-calling this funcion when the timer fires the COMPLETE event. 2. If the flashvar jsInterfaceReadyFunc is set to false, this function simply registers the KDP callbacks, without waiting for the page to report that it's ready.
