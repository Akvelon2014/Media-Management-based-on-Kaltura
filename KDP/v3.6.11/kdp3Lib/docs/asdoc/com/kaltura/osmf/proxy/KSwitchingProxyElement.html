Packagecom.kaltura.osmf.proxy
Classpublic class KSwitchingProxyElement
InheritanceKSwitchingProxyElement org.osmf.elements.ProxyElement

This class is an OSMF proxy which allows to switch 2 elements, the "main" media element and the "secondary" media element. The assumption is that the "secondary" media element is of lesser importance, like an advertising mid-roll.

secondaryElementTraitsToBlockproperty 
secondaryElementTraitsToBlock:Vector  [read-write]

Because the working assumption is that the secondary media element is a form of advertising content, it is possible to set a Vector of traits that will be blocked for this element. This way, for example, we can block the time progress of the media element so that the scrubber will not show it.
