Classpublic class FlValueParser
InheritanceFlValueParser ValueParser
ImplementsIValueParser

Helper class providing easy data collection from UIcomponents. If the source is UIcomponent or TextField, property is optional. (e.g. If you want to collect data from TextField, you don't have to specify property as "text".)

UIcomponent type(source) and default property:

TextInput : text
ComboBox : selectedItem["data"]
NumericStepper : value
LabelButton(Button,CheckBox, RadioButton) : selected
List, TileList : selectedItem[labelField]
ColorPicker : hexValue
