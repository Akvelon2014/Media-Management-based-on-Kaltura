Filtering Content in the Entries Table

The list of entries can be filtered according to the parameters that are specified in the filter bars on the left side of the page. You can filter entries using the search criteria, specific categories,  Additional Filters, or custom metadata based filters

 

NOTE: You can sort the Entries Table by clicking on the various columns or using the scroll bar. You can also select how many items to display per page (10 to 500).
