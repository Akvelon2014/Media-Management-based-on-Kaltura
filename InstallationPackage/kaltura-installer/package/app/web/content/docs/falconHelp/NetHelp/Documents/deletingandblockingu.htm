Deleting and Blocking Users

You can prevent users from logging into the KMC temporarily or permanently. Blocking a user will prevent the user from logging into the KMC. This restriction can be removed by applying the unblock action.

Deleting a user is a permanent action. The deleted user's content contribution records will be kept in the system.

  To block or unblock a user

1.   Select the Administration tab and select the Users tab.

2.   Click on a User Name.

3.   Select Block/Unblock from the Select Action drop-down menu to block a user temporarily.

4.   Select Delete from the Select Action drop-down menu to remove a user permanently.
