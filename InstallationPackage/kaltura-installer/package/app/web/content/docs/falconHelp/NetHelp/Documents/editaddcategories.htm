Edit - Add Categories

You can add categories to entries in the KMC that will propagate to other applications.

  To add categories to an entry

1.   Select the Content tab and then select the Entries tab.

2.   Select an entry or multiple entries and select Bulk Actions on the bottom of the page.

3.   Select Add Categories from the drop-down menu.

4.   Select or search for a category to apply to all the selected entries.

5.   Click Apply.
