Preview and Embed

To preview and get the embed code for entries in the Entries Table; click the Preview & Embed" link under the Actions column on the main list of entries. In the new window that opens, you can preview the entry and select the player that you would like to use.

  To preview and embed your content to a website

1.   Go to the Content tab and select the Entries tab.

2.   Click on an entry in the Entries table.

3.   Select Preview and Embed in the actions drop down menu.

4.   In the new window that opens, you can preview the entry and select the player that you would like to use.

5.   Grab the embed code.
