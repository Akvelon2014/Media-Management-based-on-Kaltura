KMC Publisher Tasks for TubeMogul Syndication

  To configure syndication settings for TubeMogul

1.   Configure the Syndication Settings. See Setting Up Syndication.

2.   Select TubeMogul from the Feed Type drop-down menu.

The TubeMogul parameters are displayed.

3.   Enter the TubeMogul Parameters, see TubeMogul: Feed Parameters and click Add Feed.

The URL for the feed is generated.
